##Summer有哪些功能？

* 方便的`数据库`支持
    *  无需关注底层数据库交互，统一CRUD执行方法
    *  支持SQLServer、MYSQL、ORACLE、DB2、Sybase
* 带有`市场插件`，免费在市场安装所需组件
* `开发便捷`支持，方便PHP新手快速上手
* 强大的`路由`功能，方便定制自己的路由
* 有数量也有质量的`主题`,编辑器和预览区域
* 完美兼容`Github`的markdown语法
* 代码编辑预览区域`代码高亮`
* 所有选项自动记忆
